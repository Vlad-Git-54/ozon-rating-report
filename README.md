# Ozon User Rating Report
Автор - Марьяновский Владислав Андреевич

Проект генерирует Excel-отчёт по товарам магазина Ozon:

- SKU
- Offer ID
- Наименование товара
- Средний пользовательский рейтинг (средний по отзывам покупателей, без сложной логики)
- Количество отзывов
- Дата последнего отзыва
