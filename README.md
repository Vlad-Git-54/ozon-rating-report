# Ozon User Rating Report
Автор - Марьяновский Владислав Андреевич

Проект генерирует Excel-отчёт по товарам магазина Ozon:

- SKU
- Offer ID
- Наименование товара
- Средний пользовательский рейтинг (по отзывам)
- Количество отзывов
- Дата последнего отзыва

Отчёт строится через Ozon Seller API и, при желании, отправляется в Telegram-чат.
